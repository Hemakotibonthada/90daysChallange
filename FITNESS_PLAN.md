# 90-Day Physical Fitness Plan

## Goal
Build a sustainable fitness habit. Progress from basic walking to a complete workout routine.

---

**Focus:** Build the habit, get moving daily

|-----|----------|----------|
| Mon | Brisk walk + full body stretch | 30 + 5 min |
| Tue | Brisk walk + full body stretch | 30 + 5 min |
| Wed | Brisk walk + full body stretch | 30 + 5 min |
| Thu | Brisk walk + full body stretch | 30 + 5 min |
| Fri | Brisk walk + full body stretch | 30 + 5 min |
| Sat | Brisk walk + full body stretch | 30 + 10 min |

### Week 2: Walking + Bodyweight Basics
| Day | Activity | Details |

| Tue | Walk + Squats | 30 min walk + 3x10 bodyweight squats |

### Week 3-4: Light Jogging + Progressive Bodyweight
| Day | Activity | Details |
|-----|----------|---------|
| Mon | Jog/walk intervals + Upper body | 20-25 min + push-ups (3x10), arm circles |
| Tue | Jog/walk intervals + Lower body | 20-25 min + squats (3x15), lunges (3x10) |

| Fri | Jog + Flexibility | 25 min jog + 10 min stretching |
| Sat | Jog + Full body | 25 min jog + full bodyweight routine |
| Sun | Rest / Yoga/stretching | 20-30 min |

## Phase 2: Building Strength (Day 31–60)
**Focus:** Increase intensity, build strength & endurance

### Weekly Template
| Day | Focus | Details |
|-----|-------|---------|
| Mon | Run + Upper Body | 25-30 min run + push-ups (3x15), dips (3x8), arm exercises |
| Tue | Run + Lower Body | 25-30 min run + squats (3x20), lunges (3x12), calf raises |
| Wed | HIIT + Core | 20-25 min HIIT + planks (3x30s), leg raises, mountain climbers |
| Thu | Run + Full Body | 30 min run + compound exercises circuit |
| Fri | HIIT + Strength | 25 min HIIT + progressive overload on all exercises |
| Sat | Long Run + Flexibility | 30-35 min run + 10-15 min deep stretching |
| Sun | Active Rest | Light walk + yoga/stretching |

### Progressive Overload Schedule
| Exercise | Week 5 | Week 6 | Week 7 | Week 8 |
|----------|--------|--------|--------|--------|
| Push-ups | 3x12 | 3x15 | 4x15 | 4x18 |
| Squats | 3x15 | 3x20 | 4x20 | 4x25 |
| Planks | 3x20s | 3x30s | 3x40s | 3x45s |
| Lunges | 3x10 each | 3x12 each | 3x15 each | 4x15 each |
| Burpees | 3x5 | 3x8 | 3x10 | 3x12 |
| Mountain Climbers | 3x10 | 3x15 | 3x20 | 3x25 |
| Running Distance | 3 km | 3.5 km | 4 km | 4.5 km |

### HIIT Workout (20-25 min)
```
Warm-up: 3 min light jog
Round 1-4 (repeat 4x):
  - 30s Jumping Jacks
  - 30s Rest
  - 30s Burpees
  - 30s Rest
  - 30s Mountain Climbers
  - 30s Rest
  - 30s High Knees
  - 60s Rest between rounds
Cool-down: 3 min walk + stretch
```

## Important Notes

1. **Warm up** before every workout (5 min light cardio + dynamic stretching)
2. **Cool down** after every workout (5 min walk + static stretching)
3. **Hydration:** Drink at least 2-3 liters of water daily
4. **Sleep:** Get 7-8 hours of sleep
5. **Nutrition:** Eat balanced meals, prioritize protein
6. **Listen to your body:** Rest if you feel pain (not just soreness)
7. **Consistency > Intensity:** It's better to do a light workout than skip entirely
