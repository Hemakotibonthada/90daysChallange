# Hema Koteswar Naidu's 90-Day Challenge

**Start Date:** April 17, 2026  
**End Date:** July 16, 2026  
**Goal:** Master C++, Python, DSA & Physical Fitness

---

## Challenge Rules

1. Code **every single day** — no zero days
2. Minimum **2-3 hours** of coding daily
3. Minimum **30-45 minutes** of physical fitness daily
4. Track progress in the daily checklist
5. Push code to this repo every day
6. Solve at least **1 DSA problem** daily (from Day 10 onwards)

---

## 3-Phase Structure

### Phase 1: Foundations (Day 1–30)
| Week | C++ | Python | DSA | Fitness |
|------|-----|--------|-----|---------|
| Week 1 | Setup, syntax, variables, I/O, operators | Setup, syntax, variables, I/O, operators | Big-O notation, time/space complexity | Walking 30 min + stretching |
| Week 2 | Conditionals, loops, functions | Conditionals, loops, functions | Arrays & Strings | Walking 30 min + bodyweight basics |
| Week 3 | Arrays, pointers, references | Lists, tuples, dictionaries, sets | Linked Lists (singly & doubly) | Light jogging + push-ups/squats |
| Week 4 | Strings, STL basics (vector, map) | String manipulation, list comprehensions | Stacks & Queues | Jogging 20 min + core exercises |

### Phase 2: Intermediate (Day 31–60)
| Week | C++ | Python | DSA | Fitness |
|------|-----|--------|-----|---------|
| Week 5 | OOP (classes, objects, constructors) | OOP (classes, inheritance, polymorphism) | Trees (binary tree, BST) | Running 25 min + full body workout |
| Week 6 | Inheritance, polymorphism, abstraction | File I/O, exception handling | Tree traversals, heap/priority queue | Running + progressive overload |
| Week 7 | Templates, operator overloading | Decorators, generators, iterators | Hashing & Hash Maps | HIIT 20 min + strength training |
| Week 8 | STL deep dive (algorithms, iterators) | Modules, packages, virtual environments | Graphs (BFS, DFS) | HIIT + compound exercises |

### Phase 3: Advanced (Day 61–90)
| Week | C++ | Python | DSA | Fitness |
|------|-----|--------|-----|---------|
| Week 9 | Memory management, smart pointers | Advanced OOP, design patterns | Graph algorithms (Dijkstra, Topological Sort) | Full workout routine 45 min |
| Week 10 | File I/O, exception handling | Regular expressions, lambda functions | Dynamic Programming (1D) | Full workout + flexibility |
| Week 11 | Multithreading basics, move semantics | List/dict comprehensions mastery, map/filter | Dynamic Programming (2D), backtracking | Full workout + endurance |
| Week 12-13 | Mini project in C++ | Mini project in Python | Mock interviews, contest problems | Maintain routine + personal records |
